# Dia 15 

Um programa que le dois números inteiros positivos 
e verifica se são números amigos. 

📌 O que são números amigos?

Dois números são amigos quando:

* A soma dos divisores próprios do primeiro número = segundo número
* E a soma dos divisores próprios do segundo número = primeiro número

👉 Exemplo:

* 220 → divisores próprios: 1,2,4,5,10,11,20,22,44,55,110 → soma = 284
* 284 → divisores próprios: 1,2,4,71,142 → soma = 220

- loops for
- condições if (n % i == 0)
